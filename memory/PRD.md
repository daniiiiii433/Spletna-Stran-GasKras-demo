# Gas Kras Band Website - PRD

## Original Problem Statement
Sito web professionale per la band musicale Gas Kras - versione demo/scheletro con:
- Design light/pulito con approccio folk-friendly
- Multilingua: Sloveno, Italiano, Inglese
- Solo frontend (no backend)
- Placeholder per foto, logo, prodotti merch
- Link social reali (Instagram, Spotify, YouTube, Facebook)
- Niente form contatto - solo link social

## User Personas
1. **Fan della band**: Vuole ascoltare musica, vedere video, seguire news
2. **Promoter/Booking Agent**: Cerca contatti, bio, materiale promozionale
3. **Stampa/Media**: Cerca materiale press, foto, info band
4. **Potenziale cliente merch**: Vuole acquistare merchandise

## Core Requirements (Static)
- 8 pagine: Home, Music, Videos, About, Gallery, Merch, News, Contact
- Navigazione responsive con menu mobile
- Switch lingua SLO/ITA/ENG
- Integrazione embed Spotify/YouTube
- Link social funzionanti
- Design "Organic & Earthy" con palette terracotta/stone
